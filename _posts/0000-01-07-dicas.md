## Dicas Finais

---

### .gitignore — ignore o que não deve ir ao repo

```
# Python
__pycache__/
*.pyc
venv/

# Node
node_modules/

# Segredos — NUNCA commitar
.env
*.key
```

---

### Mensagens de commit que comunicam

```bash
# Ruim
git commit -m "fix"
git commit -m "alterações"

# Bom
git commit -m "Corrige cálculo de total quando desconto é zero"
git commit -m "Adiciona autenticação JWT na API de usuários"
```
