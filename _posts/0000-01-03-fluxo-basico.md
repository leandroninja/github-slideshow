## Fluxo Básico do Git

```
Working Directory → Staging Area → Repositório Local → Repositório Remoto
```

---

### Comandos essenciais

| Comando | O que faz |
|---------|-----------|
| `git init` | Inicializa repositório |
| `git clone <url>` | Copia repositório remoto |
| `git status` | Mostra estado atual |
| `git add <arquivo>` | Adiciona à staging area |
| `git commit -m ""` | Salva snapshot |
| `git push` | Envia para o remoto |
| `git pull` | Baixa e integra alterações |
