## O que é Git?

Git é um **sistema de controle de versão distribuído** criado por Linus Torvalds em 2005.

---

### Por que usar?

- Histórico completo de todas as mudanças
- Trabalho em equipe sem conflito
- Reversão fácil de erros
- Ramificação (branches) para features isoladas

---

```bash
# Os três comandos mais usados no dia a dia
git add .
git commit -m "mensagem clara do que foi feito"
git push origin main
```
