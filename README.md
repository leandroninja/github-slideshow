# Git & GitHub na Prática

Apresentação interativa sobre fundamentos de Git e GitHub, construída com [reveal.js](https://github.com/hakimel/reveal.js/) e Jekyll.

## Slides

1. Introdução
2. O que é Git?
3. Fluxo básico (add → commit → push)
4. Branches e isolamento de features
5. Pull Requests e revisão de código
6. GitHub Actions — CI/CD
7. Dicas finais e boas práticas

## Como rodar localmente

```bash
gem install bundler
bundle install
bundle exec jekyll serve
```

Acesse `http://localhost:4000`

## Navegação

| Tecla | Ação |
|-------|------|
| `→` / `Space` | Próximo slide |
| `←` | Slide anterior |
| `F` | Tela cheia |
| `Esc` | Visão geral |
| `S` | Notas do apresentador |
